# Rule-based Internship Help Chatbot
## Project Overview
This project is a rule-based Internship Help Chatbot developed using Python.  
The chatbot helps students by providing information about internships, required skills, application process, benefits, and interview preparation.  
It works by matching user input with predefined keywords and rules.
Objective of this project is 
- To build a simple chatbot using rule-based logic.
- To understand basic natural language processing.
- To learn conversation flow handling using Python.
## Technologies Used
- Python 
- If-else statements
- Keyword-based pattern matching
## How the Chatbot Works
- User enters a message.
- Input is converted to lowercase.
- Keywords are checked using if-else conditions.
- Corresponding response is displayed.
## How to Run the Project
1. Open terminal or command prompt in the project folder.
2. Run the command:python chatbot.py
3. Start chatting with the bot.
4. Type `bye` to exit the chatbot.
## Output
- The chatbot runs in the terminal and interacts with the user.
- It provides internship-related responses based on user input.
- The program exits when the user types bye.
## Features
- Greeting responses
- Internship information
- Skills guidance
- Application process details
- Interview tips
- Exit option
